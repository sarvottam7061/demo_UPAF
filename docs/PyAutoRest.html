class PyRest (url=None)

A Requests session.

Provides cookie persistence, connection-pooling, and configuration.

Basic Usage::

import requests s = requests.Session() s.get('https://httpbin.org/get')

Or as a context manager::

with requests.Session() as s: … s.get('https://httpbin.org/get')

Creates a rest_client to access different rest methods. Acts as a wrapper for requests module

Args:
    url: string -> not mandatory

Ancestors

  • requests.sessions.Session
  • requests.sessions.SessionRedirectMixin
